# IBM Applied Data Science Capstone

This repository contains all notebooks, scripts, and outputs for the IBM Applied Data Science Capstone Project.

## Project Overview
The objective of this project is to analyze SpaceX Falcon 9 launch data and predict first-stage landing success using data collection, EDA, visualization, and machine learning classification models.

## Repository Structure
- Data Collection (API & Web Scraping)
- Data Wrangling
- Exploratory Data Analysis (EDA)
- Interactive Visual Analytics (Folium & Plotly Dash)
- Machine Learning Prediction
- Screenshots and Outputs

## Files
- jupyter-labs-spacex-data-collection-api.ipynb
- jupyter-labs-webscraping.ipynb
- labs-jupyter-spacex-Data wrangling.ipynb
- edadataviz.ipynb
- lab_jupyter_launch_site_location.ipynb
- spacex-dash-app.py
- SpaceX_Machine_Learning_Prediction_Part_5.ipynb

## Dashboard
A Plotly Dash dashboard is included to visualize launch success rates and payload relationships.
